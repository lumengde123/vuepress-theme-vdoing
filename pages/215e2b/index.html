幻影星空-故障维修笔记

本文列举了幻影星空软件和设备常见故障及解决方法

# 常见故障和解决方法

阅读设备常见故障及解决办法，如遇到无法解决的问题，请及时在服务群联系技术人员。

# 游戏播放时异常

  • 游戏没声音：三分钟内按键盘左上角ESC键停止，重新开一下游戏。如果超过三分钟，可以录像取证备份，打开F1->点播记录，拍照，在服务群退星点（游戏时间点+本店ID）
  • 眼镜没画面：三分钟内重启游戏，还是不行先换其他座位营业，闲了联系服务群技术人员/会弄的话，可以自己打开大朋助手查看眼镜状态

...

# XDPlayer播放器提示：试用结束

请按上图这样拍照，露出底下的店铺ID，发送到飞碟服务群并@本店的技术人员处理

# XDPlayer播放器提示：急停

一般双子星会有这个提示。检查一下：

  • 两个座位的横向安全带是否扣上
  • 机器背后红色按钮是否点亮
  • 电脑右下方急停按钮不能被压下

# XDPlayer播放器提示：网络已禁用/网络未连接

如果提示网络已禁用，可能是使用完XD Control（XD动感控制），网络控制忘记点是

如果是网络未连接，可能是电脑没连接上网线/wifi，也可能是没网费了

# 显示器黑屏并提示 No Signal

  • 建议：直接问服务群里技术人员，自己别尝试
  • 可能是哪里松了，或者死机了需要换主机

# 提示：当前设备 token 已绑定其他设备，请重新设置

  • 看下是不是机器后面开关没启动，机器和座位都开机后就没提示了
  • 如果不行，就问群里技术支持解决吧

# VR眼镜黑屏 看不见画面

可能是眼镜出了点故障，重新连接一下就好
打开TeamViewer，伙伴ID按顺序分别对应2~6号座位，点击连接，密码默认123456

连接上之后在黑窗口中打开大朋助手，点击重置头戴式显示器就行了，然后关闭黑窗口
如果是1号座位，不用登录TeamViewer，直接在电脑中打开大朋助手来重置

# 安全带卡扣松动

在工具箱找到内六角扳手套装，拿出第一排最小的那个

把卡扣调整到合适的角度，长边抵住螺丝凹槽，转几圈拧紧螺丝即可

# 游戏结束扫腿还一直喷气

可能是座位下面的电磁阀有问题，先咨询技术人员

如果要求关闭气泵，它放在小楼梯下面

在气泵侧边有个红色按钮（盲摸一下能找到），按下去可以关闭气泵，不再喷气扫腿。等待几天后技术人员来现场更换。这个关闭不会影响设备摇晃，对游戏体验影响不大。

补充：如果喷气了，但是扫腿的绳子不动，考虑是不是卡在夹板之间的缝隙里了。

# 飞碟倾斜没复位

  • XDPlayer播放器中，按F1->选择复位
  • 影片播放时如果机器能动，则驱动器和控制卡状态是好的，继续尝试F1进行复位。如果播放时也不动则需要查看驱动器状态，VR控制器选择P0-66，如果值为2000就调成1000 【幻影星空 | VR控制器 P0-66】 (opens new window)。如果本来就1000 那就是其他问题
  • 设备服务群咨询

# 战舰特效灯一闪一闪的

有可能是12伏电源短路导致，需要更换电源

在XD Control中，设置前后倾斜-最高，让设备抬头。然后撕开下面的黑布（底部是贴纸可以打开）

左边两个白色盒子是24伏电源，和电脑有关；右上角是12伏电源，和灯有关

如果12伏电源的灯也一闪一闪的，说明短路了，需要更换220V 12伏 350W 的开关电源

先去服务群里问一下，等技术人员来更换就行

# 开关机按钮(红色绿色)有时失灵

通常是按钮后面的卡扣松了，需要调一下

  1. 确保整个设备断电。如果按钮还亮的说明没成功断电，设备是没有备用电源的。

  2. 用螺丝刀拆开按钮上面的大盖板，斜靠在自己身上

  3. 一只手抵住按钮后面，另一只手按按钮，多按几下

  4. 盖板装回去，通电，开关机2~3次看看按钮是否流畅

# 常用操作

# 星点充值

在 XD-Player 播放器右上角，有星点充值的按钮，打开选择对应的套餐，扫码付款就行了

# TeamViewer - 连接其他座位主机

我们平时在电脑上操作的，是1号主机，对应1号座位，也控制着整个机器。如果需要调整其他座位的眼镜，需要打开其他座位的主机，那就要使用TeamViewer来操作

  • 打开桌面上的TeamViewer软件

  • 这里的几个ID按顺序，依次代表2号~6号座位的主机

  • 例如：选择192.168.10.5，就代表打开4号座位的主机
  • 登录密码默认是 123456

  • 点击登录后可以看到这个窗口，这里就是4号主机电脑桌面
  • 在这里打开大朋助手或者SteamVR等软件，对应的就是4号座位的眼镜

# 大朋助手 - 查看眼镜状态

  • 双击打开桌面红色的大朋助手，可以看到主机（1号座位）的VR眼睛状态
  • 如果眼镜全黑，打开这个软件会弹出提示，重新加载还是什么(忘记截图了)，然后眼镜就恢复正常了
  • 同样的，如果要调整其他座位的眼镜，需要先通过TeamViewer远程连接一下，在远程桌面里打开大朋助手

# XD动感控制 - 设备倾斜和特效

  • 在电脑右下角可以找到一个空心的三角形图标 XDControl，双击打开（如果从桌面打开，不会自动显示软件，要从右下角打开）。首先将网络控制点否

  • **设备倾斜：**然后在姿态运动测试里可以控制设备的倾斜姿态，例如：前后倾斜+最高，然后点击运行，可以看到战舰/飞碟 就会抬头倾斜。如果不用了，就点击绿色的回原位，

  • 特效测试：在下面点击端口测试，选择特效测试，然后可以勾选扫腿、捶背等测试座椅功能是否正常

  • 最后关闭软件前，把网络控制点是就行了。【如果发现XDPlayer没网络/网络已禁用，可能是网络控制忘了点是】

# XD播放器 - 查看星点

点击播放器右上角的星点充值按钮，在弹出来的窗口可以看到星点余额，如果低于1500，及时叫老板充值

# XD播放器 - 影片下载暂停删除

注意

设备一次只能下载一个影片，并且影片在下载和分发(安装)时不能播放游戏

如果有顾客，需要在XD Update里暂停下载/分发，再播放游戏

如果新游戏播放异常，考虑

一、影片下载：

  • 在XD播放器中，点击左上角的商店，选择影片，不用输入密码直接点确定，就可以进入影片商店

  • 在右侧可以输入游戏的名字，按Enter跳转（如果不想筛选游戏，要把输入框清空）

  • 点击一个游戏，可以看到它的影片预览、影片简介、游戏时长等信息。右下角点击下载就可以开始下载游戏了

下载注意事项

所有游戏下载的时候请注意：

  1. 切记！！！下载游戏是必须所有的座位主机头盔正常亮起，代理软件全部打开才可以下载。
  2. 下载游戏的过程中 设备不可以播放影片 否则设备会瘫痪崩溃的
  3. 如果有客人要玩 就直接暂停下载 等玩完以后再继续下载即可。
  4. 下载游戏的同时 只能一个影片下载完成以后，再点另外一个下载 否则会拖垮网速卡死的
  5. 全部下载完成后 重启播放器即可

二、影片暂停下载：

  • 如果下载过程中有顾客要玩，需要暂停下载/分发。在右下角打开XD Update（XD更新），在弹出的窗口里找到正在下载的游戏

  • 点击右侧按钮进行暂停。然后可以在XD播放器里放游戏了。
  • 结束之后，在点击按钮继续下载/分发

安装完成后，需要关闭和重启播放器，这样新游戏才能使用。

提示

如果重启完播放器，新游戏还是播放异常，考虑播放一下相关插件，自动更新/下载 一些必备文件

相关教程：幻影星空 | 新游必备插件安装教学 (opens new window)

三、影片删除：

在XD播放器中，按F1，选择影片信息，不输入密码直接点击确定，找到要删除的影片，点击删除即可。

如果影片故障要重新下载，需要先删除，再下载。

# VR眼镜 - 拆卸和更换

幻影星空的VR眼镜使用的是大朋家的DPVR E3-C，点击查看【E3-C 文档 | 英文版】 (opens new window)

拆卸视频教程：幻影星空 | DPVR E3-C 拆卸教程 (opens new window)

# 射击按钮 - 检查失灵与更换

  • 假设3号座位的射击按钮失灵了，那就先打开TeamViewer登录到3号座位的主机（弹出一个电脑桌面，背景是黑色的）

  • 打开桌面上的控制面板，选择查看设备和打印机

  • 右键XD5D，选择游戏控制器设置

  • 点击属性，可以看到下图右侧这个窗口

按下3号座位的按钮，这里的按钮图标其中一个会变亮，说明按钮正常。如果都不亮，说明按钮有故障。

如果需要更换按钮 -> 射击按钮更换教程：【幻影星空 | 射击按钮更换】 (opens new window)

用相同的方法，可以检查所有座位的按钮是否失灵
