咸虾米壁纸 uniapp

# 咸虾米壁纸项目

提示

该项目是移动端的壁纸类项目，用到的技术栈是：Uniapp + Vue3 + uView + ES6，视频链接：https://www.bilibili.com/video/BV1ie411f7e6/ (opens new window)

# 项目简介

项目名称： 咸虾米壁纸

项目介绍和 API 接口： https://api.qingnian8.com/apis/wallpaper/

课程视频： https://www.bilibili.com/video/BV1Yg4y127Fp/?spm_id_from=333.337.search-card.all.click&vd_source=147cc6eb39eb565f36bd69ce0b3069bd

软件下载： VSCode、HBuilder (opens new window)、微信开发者工具 (opens new window)

项目地址： https://gitee.com/qingnian8/univue3/tree/master

需要在HBuilder->运行->运行到小程序模拟器->运行设置里，填写好微信开发者工具文件夹位置才能打开

如果微信小程序打开项目白屏，要在开发者工具->设置->通用设置->安全里打开服务端口，重新启动一下

uView： https://uviewui.com/components/color.html

uniapp官网： https://zh.uniapp.dcloud.io/component/view.html
